\doxysubsection{Detailed Description}
A cache value that contains a map of strings to strings. 

This class inherits from the {\ttfamily \mbox{\hyperlink{classContainerCacheValue}{Container\+Cache\+Value}}} base class and encapsulates a map of strings. It provides an implementation of the {\ttfamily get\+Value}, {\ttfamily set\+Value}, {\ttfamily size}, {\ttfamily contains}, {\ttfamily empty}, and {\ttfamily clear} methods. The {\ttfamily add\+Value\+To\+Redis\+\_\+} method, which is used to add the map to a Redis database, is also overridden in this class. Additional methods for manipulating the map ({\ttfamily add\+Key}, {\ttfamily erase\+Key}, {\ttfamily get\+Key}) are also provided.
