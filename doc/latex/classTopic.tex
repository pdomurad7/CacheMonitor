\doxysubsection{Detailed Description}
A class that represents a topic in the cache. 

This class maintains a set of parameters that have changed ({\ttfamily changed\+\_\+parameters\+\_\+}), a map of cache values ({\ttfamily cache\+\_\+values\+\_\+}), and the path of the topic ({\ttfamily topic\+\_\+path\+\_\+}). \mbox{\hyperlink{classTopic}{Topic}} path is path where values from this topic are stored in redis. It provides methods for checking and clearing changed parameters, adding, removing, and accessing cache values, and checking if a cache value exists. It also provides methods for managing the topic path.
