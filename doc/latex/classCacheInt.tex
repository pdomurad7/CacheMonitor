\doxysubsection{Detailed Description}
A cache value that contains an integer. 

This class inherits from the {\ttfamily \mbox{\hyperlink{classAbstractCacheValue}{Abstract\+Cache\+Value}}} base class and encapsulates an integer value. It provides an implementation of the {\ttfamily get\+Value} method, which retrieves the integer value as a {\ttfamily std\+::any} object, and the {\ttfamily set\+Value} method, which sets the integer value. The {\ttfamily add\+Value\+To\+Redis\+\_\+} method, which is used to add the value to a Redis database, is also overridden in this class.
