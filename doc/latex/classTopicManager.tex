\doxysubsection{Detailed Description}
A singleton class that manages a collection of {\ttfamily \mbox{\hyperlink{classTopic}{Topic}}} objects. 

This class provides methods for creating, removing, and accessing {\ttfamily \mbox{\hyperlink{classTopic}{Topic}}} objects. It maintains a map of topic names to {\ttfamily \mbox{\hyperlink{classTopic}{Topic}}} pointers. The {\ttfamily get\+Instance} method is used to access the single instance of this class.
