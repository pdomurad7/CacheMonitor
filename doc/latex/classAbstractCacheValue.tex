\doxysubsection{Detailed Description}
Abstract base class for cache values. 

Base cache value class, which provides common interface for manipulating cache values, like getting their values, adding values to redis or changing topic. This functionalities are needed in class like \mbox{\hyperlink{classTopic}{Topic}} or \mbox{\hyperlink{classTopicManager}{Topic\+Manager}}.
