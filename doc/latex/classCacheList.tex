\doxysubsection{Detailed Description}
A cache value that contains a list of strings. 

This class inherits from the {\ttfamily \mbox{\hyperlink{classContainerCacheValue}{Container\+Cache\+Value}}} base class and encapsulates a list of strings. It provides an implementation of the {\ttfamily get\+Value}, {\ttfamily set\+Value}, {\ttfamily size}, {\ttfamily contains}, {\ttfamily empty}, and {\ttfamily clear} methods. The {\ttfamily add\+Value\+To\+Redis\+\_\+} method, which is used to add the list to a Redis database, is also overridden in this class. Additional methods for manipulating the list ({\ttfamily rpush}, {\ttfamily lpush}, {\ttfamily rpop}, {\ttfamily lpop}) are also provided.
