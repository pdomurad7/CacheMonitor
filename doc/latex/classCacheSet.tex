\doxysubsection{Detailed Description}
A cache value that contains a set of strings. 

This class inherits from the {\ttfamily \mbox{\hyperlink{classContainerCacheValue}{Container\+Cache\+Value}}} base class and encapsulates a set of strings. It provides an implementation of the {\ttfamily get\+Value}, {\ttfamily set\+Value}, {\ttfamily size}, {\ttfamily contains}, {\ttfamily empty}, and {\ttfamily clear} methods. The {\ttfamily add\+Value\+To\+Redis\+\_\+} method, which is used to add the set to a Redis database, is also overridden in this class. Additional methods for manipulating the set ({\ttfamily add\+Value}, {\ttfamily remove\+Value}) are also provided.
