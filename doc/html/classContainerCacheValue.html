Detailed Description

An abstract cache value that contains a container.

This class inherits from the AbstractCacheValue base class and serves as the base class for cache values that contain containers such as lists, maps, or sets. It declares several pure virtual functions that must be implemented by the derived classes: clear, size, contains, and empty.
