Detailed Description

A cache value that contains a float.

This class inherits from the AbstractCacheValue base class and encapsulates a float value. It provides an implementation of the getValue method, which retrieves the float value as a std::any object, and the setValue method, which sets the float value. The addValueToRedis_ method, which is used to add the value to a Redis database, is also overridden in this class.
